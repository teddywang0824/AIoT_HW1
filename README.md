## 📋 專案概述

本專案為 AIoT 課程第一次作業，建立一個具有現代設計感的**個人介紹網頁**，並即時顯示當前時間。頁面採用深色主題搭配動態粒子背景，呈現出專業且富有科技感的視覺效果。

## ✨ 功能特色

| 功能 | 說明 |
|------|------|
| 🧑‍💻 個人介紹 | 顯示姓名、身份標籤與簡介 |
| ⏰ 即時時鐘 | 每秒更新的數位時鐘，含完整日期與星期 |
| 🌌 粒子動態背景 | 星空般的粒子連線動畫效果 |
| 💎 玻璃擬態設計 | 毛玻璃半透明卡片與漸層色彩 |
| 📱 響應式設計 | 自動適配桌面版與行動裝置 |
| 🎯 滾動進場動畫 | 元素進入視窗時的淡入動畫效果 |

## 🛠️ 技術架構

- **HTML5** — 語意化標籤結構
- **CSS3** — 自訂設計系統（CSS Variables）、玻璃擬態、漸層、動畫
- **JavaScript (ES6+)** — 即時時鐘、Canvas 粒子系統、Intersection Observer API

## 📁 檔案結構

```
AIoT_HW1/
├── index.html    # 主頁面 HTML 結構
├── style.css     # 完整 CSS 設計系統與響應式樣式
├── script.js     # 時鐘更新、粒子動畫、互動邏輯
└── README.md     # 專案說明文件
```

## 🚀 使用方式

直接在瀏覽器中開啟 `index.html` 即可檢視頁面。

```bash
# 或使用任意 HTTP 伺服器
npx http-server . -p 8080
```
